# Explorely

A full-stack web application to share and discover tourist spots by state and city, with Google Maps integration and a live visitor counter. Built with React, Node.js, Express, and MongoDB.

## Features
- Browse tourist spots by state and city
- Google Maps links for each spot
- Responsive design for mobile and desktop
- Visitor counter (corner circle)
- Beginner-friendly codebase

## Getting Started

### Backend
1. Install dependencies:
   ```bash
   cd backend
   npm install
   ```
2. Set up MongoDB (local or Atlas) and update `.env` if needed.
3. Start the server:
   ```bash
   npm run dev
   ```

### Frontend
1. Install dependencies:
   ```bash
   cd frontend
   npm install
   ```
2. Start the React app:
   ```bash
   npm start
   ```
this is a complete full stack project using js, react, node.js and mangoDB


## Folder Structure
- `backend/` — Node.js/Express API and MongoDB models
- `frontend/` — React app (responsive UI)
